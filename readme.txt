Скрипт виводить ім'я пульту на сервері CASL Cloud, якому належить введений номер приладу.
По замовчуванню відображає імена ПЦС на латиниці.
1. Скопіювати файл у папку db-driver.
2. Запустити скрипт.
3. Ввести номер ппк.

Якщо ім'я ПЦС у базі на кирилиці: 
 - виведений програмою текст скопіювати в текстовий редактор AkelPad;
 - внизу вікна AkelPad правий клік, нажати "Redetect";
 - у випадаючому вікні обрати "From 866 (ОЕМ - русская) To 65001 (UTF-8)", нажати ОК.
